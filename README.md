# Mini CRM

A simple Customer Relationship Management (CRM) system built to manage customer data, interactions, and basic business workflows.

---

## 🚀 Features

* Add, edit, and delete customer details
* Manage customer interactions
* Backend API for data handling
* Clean and simple UI
* Scalable structure for future improvements

---

## 🛠️ Tech Stack

**Frontend:**

* HTML / CSS / JavaScript *(or React if you used it)*

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/Abhijeetya01/Mini_Crm.git

### 2. Navigate to the project folder

cd Mini_Crm

### 3. Install dependencies

npm install

### 4. Run the project

npm start

---

## 🔑 Environment Variables

Create a `.env` file in the root and add:

MONGO_URI=your_mongodb_connection_string
PORT=5000
